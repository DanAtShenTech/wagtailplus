Welcome to Wagtail Plus's Documentation!
=======================================

Wagtail Plus is a collection of independent modular add-ons for `Wagtail CMS <https://github.com/torchbox/wagtail>`_.

Installation
------------

Wagtail Plus can be installed via ``pip`` using the following command::

    pip install wagtailplus
